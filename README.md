# Ifconfig IP Copier (Chrome Extension)

A lightweight Chrome extension that shows your public IP and copies it to clipboard.

## Features

- Fetches public IP from `https://ifconfig.me/ip`
- Shows IP in extension tooltip
- On click, opens popup and immediately displays/copies the IP

## Install locally

1. Open `chrome://extensions/`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select this repository folder
